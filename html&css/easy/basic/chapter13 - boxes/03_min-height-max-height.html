펜더 머스탱(Fender Mustang)

펜더 머스탱은(Fender Mustang)은 1964년에 발매된 학생 모델 기타의 일환으로 50년대 학생을 위한 저렴하고 연주하기 쉬운 컨셉으로 소개 되었던 뮤직마스터(Musicmaster)와 듀오소닉(Duo-Sonic)의 진화된 모델이다. 하지만 보급형 모델 라인 자체가 없어지면서 다른 보급형 모델과 함께 단종됐다.

펜더 스트라토캐스터(Fender Stratocaster)

약칭 '스트랫(Strat)'. 1954년에 발매되어 세 개의 싱글코일 픽업과 싱크로나이즈드 트레몰로를 탑재하고 있다는 것이 당시에는 화재가 되었다. 60년대 이전까지 네크와 자판이 하나로 붙어 있는 원피스형태를 취하다가 62년형 빈티지 시리즈부터 투피스 형태를 취하기 시작했다.

깁슨 레스폴(Gibson Les Paul)

현재 펜더와 함께 1,2위를 다투는 메이커가 된 깁슨(Gibson) 사에서 양산한 레스폴(Les Paul)은 스트랫과 더불어 가장 많이 사용되는 일렉트릭 기타로 유명하다. 밀도 높은 목재와 조립법을 사용하여 긴 서스테인과 묵직환 저음 드라이브 톤이 매력적이다. 사운드는 묵직해서 메탈에서 많이 사용된다.
